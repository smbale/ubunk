# Ubunk

The high-stakes myth-debunking arena where truth hits hard. Knockout the nonsense, climb the weight classes, and join the Hall of Fame.

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/167b7c36-f750-4d72-be52-7dc3ad115d7c

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
